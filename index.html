Privacy Policy

Effective Date: April 16, 2026

Overview

This browser extension is designed to improve browsing safety and focus by filtering or blocking specific search queries based on user-defined rules. It is intended for personal, family, or managed-use environments.

Data Collection

This extension does not collect, store, transmit, or sell any personal data.

  • No browsing history is collected
  • No keystrokes are recorded
  • No search queries are transmitted to external servers
  • No personally identifiable information is gathered

All processing occurs locally within the user’s browser.

Data Usage

All filtering logic is performed locally on the user’s device. Any configuration files (such as keyword lists) are used only to determine whether specific browsing actions should be allowed or blocked.

No data is sent to external servers.

Data Sharing

This extension does not share any data with third parties.

Third-Party Services

This extension does not use analytics services, advertising networks, or external tracking tools.

Permissions

The extension may request limited browser permissions in order to:

  • Read the current page URL to apply filtering rules
  • Execute content scripts to enforce blocking rules

These permissions are used solely for functionality and not for data collection.

Security

All filtering and rule processing occurs locally. No remote code execution is performed.

Changes to This Policy

This privacy policy may be updated if functionality changes. Updates will be reflected in a new version of the extension.

Contact

If you have questions about this privacy policy, you may contact the developer via the support email or listing page associated with this extension.
